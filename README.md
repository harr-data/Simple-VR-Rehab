⚡ VR Rehabilitation System: Neural Tracking Module
A high-performance web-based rehabilitation suite designed for motor-cognitive therapy. This system utilizes a "Cyber-grid" interface to provide real-time feedback and diagnostic metrics for patients undergoing neuro-rehabilitation.

⊙ Project Overview
Traditional rehabilitation can be repetitive and difficult to track. This system digitizes the recovery process by monitoring three critical areas of neuro-physical health:

Target Tracking (Motor Control): Analyzes hand-eye coordination and spatial stability.

Neural Reflex (Reaction Speed): Measures response times to dynamic stimuli in milliseconds.

Memory Sequence (Cognitive Processing): Tests short-term recall and sequential logic pattern recognition.

🛠️ Technical Architecture
Frontend: HTML5, CSS3 (Advanced animations & Scanline effects), Vanilla JavaScript.

Neural Interface Logic: Asynchronous state management to ensure zero-latency feedback during reflex testing.

Clinical Metrics: Real-time calculation of Stability Index, Maximum Deviation (px), and Neural Response Latency (ms).

🚀 Key Features
Real-time HUD: A Head-Up Display that provides instant feedback on cursor position and distance from targets.

Performance Analytics: Automated end-of-test reporting with percentage-based accuracy and stability tracking.

Interactive Design: Uses hardware-accelerated transitions and neural-themed UI/UX to maximize patient engagement.

📋 How to Use
Clone the repository: git clone https://github.com/harr-data/VR-Rehab-System.git.

Launch the interface: Open index.html in any modern web browser.

Calibrate: Select a protocol from the top menu (Tracking, Reflex, or Memory).

Initiate: Click "Initiate Test" to begin the 15-second diagnostic window.
